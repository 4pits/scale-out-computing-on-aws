# Module Overview

The workshop includes the following modules.  Each module guides you through completing a step within the workshop.

Lab | Lab Module | Description
--- | --- | ---
1 | **Deploy environment** | Using CloudFormation, provision an elastic computing environment on the AWS Cloud.
2 | **Configure Remote Desktop** | Define parameters for the remote desktop login server.
3 | **Launch Remote Desktop Session** | Launch a remote desktop session and log in using the NICE DCV remote visualization protocol.
4 | **Launch Vivado GUI** | Experience remote visualization using the Xilinx Vivado GUI.
5 | **Submit Batch Workload** | Observe the dynamic elasticity of the cloud by submitting a batch workload into the environment.
6 | **Review Analytics Dashboard** | Review environment activity in the cluster dashboard.
